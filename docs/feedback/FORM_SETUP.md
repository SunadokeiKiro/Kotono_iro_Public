# Googleフォーム 作成手順

## 1. フォームの作成

1. [Google Forms](https://forms.google.com) にアクセス
2. 「空白」をクリックして新規フォーム作成
3. [FORM_QUESTIONS.md](./FORM_QUESTIONS.md) を参照しながら質問を追加

## 2. フォームの設定

フォーム作成後、右上の **⚙️設定** から以下を確認：

### 「回答」タブ
- [ ] 回答をスプレッドシートに保存（推奨）
- [ ] 回答の編集を許可するか選択

### 「プレゼンテーション」タブ
- [ ] 進行状況バーを表示（推奨）
- [ ] 質問の順序をシャッフル（オフ推奨）

## 4. テスターへの共有方法

### オプション A: メールで共有
テスト参加者にメールでフォームURLを送信

### オプション B: アプリ内リンク
アプリ内に「フィードバックを送る」ボタンを追加し、タップでフォームURLを開く

```csharp
// Unityでの実装例
public void OpenFeedbackForm()
{
    Application.OpenURL("https://forms.google.com/xxxxx");
}
```

## 5. 質問を更新する場合

1. `FORM_QUESTIONS.md` を編集
2. Googleフォームを開いて変更を反映
3. `FORM_QUESTIONS.md` の編集履歴を更新
